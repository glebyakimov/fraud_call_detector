Распознавание мошеннических разговоров!
Задача: написать алгоритм, распознающий мошеннический разговор:
• Каждый разговор представляет собой wav файл длительностью до 2 минут.
• Время обработки одной записи не должно превышать 3 минуты(CPU+4090)
• Ограничений на используемые алгоритмы нет
• Алгоритм должен быть реализован на языке python
• Запрещается использовать платные API и библиотеки
• Тестовый скрипт должен позволять проводить тест по папке с файлами и 
возвращать результаты в формате csv: 
Название файла; label
label: 0 – разговор мошеннический 1 – разговор не мошеннический
Возможный базовый алгоритм:
1. Использовать библиотеку ASR(например vosk) для преобразования аудио->текст
2. Использовать собранный вручную(из открытых источников) словарь из слов-триггеров 
для распознавания мошеннического разговора
Используемые выборки:
 Выборка примеров для постановки задачи(samples)
 Тестовая выборка для тестирования качества работы алгоритма (не предоставляется)
Допускается использование сторонних выборок для обучения алгоритма
Задача считается решенной:
• Предоставлен алгоритм представлен в виде исходных кодов либо ссылки на репозиторий
• Подготовлена пояснительная записка в свободной форме с описанием и блок-схемой работы
алгоритма
Критериями отбора будет оригинальность предлагаемого подхода а также качества его работы на закрытой 
тестовой выборке
